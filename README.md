## About

This repository contains the personal portfolio of Hazari Tejash, a Software Engineer and AI enthusiast focused on Python, Java, Machine Learning, and full-stack development.

The portfolio is built with Next.js, Tailwind CSS, and Framer Motion. It includes a responsive hero section, project cards, work experience, education, skills, social links, and coding activity.

## Features

- Responsive layout for mobile, tablet, and desktop.
- Animated hero and section transitions.
- Featured project showcase with live links.
- Work experience, education, skills, FAQ, and social sections.
- SEO metadata and Open Graph preview configuration.

## Tech Stack

- [Next.js](https://nextjs.org/)
- [React](https://reactjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
- [TypeScript](https://www.typescriptlang.org/)

## Getting Started

### Prerequisites

Use Node.js 18 or newer.

### Installation

```sh
git clone https://github.com/Tejash880/Portfolio.git
cd Portfolio
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site locally.

## Scripts

```sh
npm run dev
npm run build
npm run lint
```

## Project Structure

```text
app/          Next.js app routes and global styles
components/   Reusable UI and portfolio sections
configs/      Portfolio content and links
public/       Images and static assets
types/        Shared TypeScript types
```
