
# Landing Page Template

A clean and customizable landing page template built using Next.js.

### Code Access

Once the payment is completed, you will receive the code via Lemon Squeezy. Follow this [YouTube video](https://youtu.be/1NvSG5Q1XnQ) to understand how to set up the project.

### 1. Install Dependencies

Run the following command to install the required dependencies:

```
npm install
```

### 2. Start the Development Server

Start the development server using:

```
npm run dev
```

### 3. Configure Your Project

Update the `configs/config.ts` file according to your project's requirements.

### 4. Deploy Your Project

After configuring the project, deploy the codebase to your preferred hosting provider.

Enjoy your new landing page!
